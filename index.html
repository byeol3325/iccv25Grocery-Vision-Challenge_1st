Abstract

We present a novel multi-modal fusion architecture that achieved first place in both Temporal Action Localization (TAL) and Spatio-Temporal Action Localization (STAL) tracks of the Amazon-sponsored Grocery Shopping Cart Action Recognition Challenge at ICCV 2025 Workshop. Our approach demonstrates significant performance improvements over existing methods through intelligent integration of temporal and spatial understanding for grocery shopping behavior analysis.

Performance Achievement

Our solution achieved approximately 2x performance improvement over the second-place solution across all evaluation metrics. Detailed performance results remain confidential as per challenge organizer requirements, but will be disclosed upon official publication of challenge results.

Problem Definition

The Grocery Shopping Cart Action Recognition Challenge focuses on the automated detection and localization of customer behaviors in retail environments. This task is critical for understanding consumer shopping patterns, inventory management, and loss prevention in modern retail systems.

Target Action Categories

Take

Return

Rummage

The challenge requires both temporal action localization (identifying when actions occur) and spatio-temporal action localization (identifying when and where actions occur with precise object-level annotations).

Methodology

Our winning approach employs a novel multi-modal fusion architecture that combines temporal action localization with spatio-temporal object detection. The system architecture consists of three main components working in synergy to achieve superior performance.

System Architecture Pipeline

Input Processing

Video sequence preprocessing and feature extraction

→
TAL Module

Multi-configuration temporal localization

→
STAL Module

Object detection and tracking with multi-modal fusion

→
Fusion Engine

integration and refinement in TAL & STAL predictions

→
Output

Final predictions of TAL & STAL

Temporal Action Localization (TAL)

  • Based on OpenTAD (CausalTAD, AdaTAD)
  • Multi-scale temporal feature extraction
  • Precise action boundary detection
  • Confidence-based prediction aggregation

Spatio-Temporal Action Localization (STAL)

  • GLEE-based object detection framework
  • SAM2 integration for precise segmentation
  • Temporal consistency through object tracking
  • Action-object association mechanisms

Results and Discussion

Our solution ranked first overall—surpassing every submission in the challenge's history—and won both the TAL and STAL tracks. The multi-modal fusion design was particularly effective in tackling occlusions, rapid movements, and complex object interactions.

Competitive Performance Analysis

Our method achieved approximately 2x performance improvement over the second-place solution across all evaluation metrics including mean Average Precision (mAP) at various IoU thresholds. The TAL configurations provided robust predictions, while the STAL component ensured precise spatial localization.
